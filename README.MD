# E-commerce Website

Un site e-commerce construit avec React. Ce projet propose une interface utilisateur dynamique et interactive pour la gestion des produits, des catégories et du panier.

## Fonctionnalités

- Navigation entre différentes catégories de produits
- Ajout et gestion du panier
- Visualisation des détails des produits
- Connexion et gestion des utilisateurs (authentification)
- Page d'authentification (login, logout)
- Intégration de fonctionnalités basiques de e-commerce

## Technologies utilisées

- **React** pour la construction de l'interface utilisateur
- **React Router** pour la gestion de la navigation
- **React Query** pour la gestion des requêtes API
- **CSS Modules** pour la gestion du style (Passage rapide sur ChakraUI)
- **TypeScript** pour un typage fort et une meilleure expérience de développement
